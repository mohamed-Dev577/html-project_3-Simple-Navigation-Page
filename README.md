# اسم المشروع 
project_3 Simple Navigation Page 
## وصف المشروع 
هذا هو ثالث مشروع في تعلم لغة HTML 
هدف هذا المشروع  هو التدريب علي بناء صفحة تنقل بسيطة وتنظيم المحتوي بشكل صحيح 

## التقنيات المستخدمة 
- HTML5 

## عناصر HTML المستخدمة 
- header 
- main 
- section 
- nav 
- a 
- h1 , h2 
- p
- ul 
- li 
- footer 

## ماذا تعلمت من هذا المشروع 
كيفية بناء هيكل  صفحة تنقل بسيطة كاملة
الفرق بين عناصر مثل 'section' و 'main'
استخدام شريط التنقل (nav) بشكل صحيح 
تنظيم المحتوي حيث يكون قابل للقراءة والفهم
